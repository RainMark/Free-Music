# Free-Music v1.3
***
## How to get it ?
` $ git clone https://github.com/RainMark/Free-Music.git `
## Install
` $ cd src/ && sudo make install `
## Uninstall
` $ cd /src/ && sudo make uninstall `
## Depends
+ GTK 2 , GTK 3
+ Gstreamer , Gstreamer bad plugins etc.
+ Complile need to gcc , make , GTK2 , GTK 3 dev library etc.
